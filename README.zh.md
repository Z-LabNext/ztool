# ztool

一个js工具库

## 特性

- [x] 表合并计算
- [x] 字典映射
- [x] 空值替换
- [x] 日期数组分割/合并
- [x] 下载文件

## 安装

```bash
# npm安装
$ npm i @zlabnext/ztool -S

# yarn安装
$ yarn add @zlabnext/ztool -S

# pnpm安装
$ pnpm add @zlabnext/ztool -S
```
